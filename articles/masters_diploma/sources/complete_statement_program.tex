\section{Программа для задачи в полной постановке}
\label{sec:complete_program}

\subsection{Общая схема вычислений}

Научным коллективом разрабатывается программа для моделирования описанной задачи \eqref{eq:electrostatic}, \eqref{eq:phi} в полной, трехмерной, постановке. Программа использует параллельные вычисления с помощью MPI (запланирована также поддержка CUDA) и предназначена для работы на суперкомпьютерных комплексах.

Кратко обозначим схему вычислений. Приведенное описание во многом следует работе \cite{zipunova_computational}.

Расчетная область $\clOmega = [0, L]_x \times [0, L]_y \hm \times [0, L]_z$ имеет форму куба; в области~$\clOmega$ вводится равномерная ортогональная расчетная сетка $\omega_h$ с шагом $h$ по каждой из~координатных осей, $M = L / h \in \Natural$~-- число шагов по пространству. По времени сетка~$\omega_h$ также равномерная, с шагом $\tau$. Целые пространственные индексы $j_{1,2,3} = \overline{0, M - 1}$ сопоставлены центрам ячеек сетки $\omega_h$; для обозначения точки на сетке используется три нижних, пространственных, индекса и один верхний, временной. Таким образом, $\omega_{j_1,j_2,j_3}^k = [(j_1 + 1/2)h \vect{e}_x + (j_2 + 1/2)h \vect{e}_y + (j_3 + 1/2)h \vect{e}_z, k \tau]$.

Свободным переменным $\phi$, $\Phi$, $\rho$ модели соответствуют сеточные функции; их значения отнесены к центрам ячеек сетки $\omega_h$. В качестве аргументов функции принимают описанный выше набор целых индексов: $\phi_{j_1,j_2,j_3}^k$ соответствует значению $\phi(\omega_{j_1,j_2,j_3}^k)$. 

Вводятся также двойственные сетки $\omega^*_\alpha$, $\alpha = 1,2,3$. Каждая $\omega^*_\alpha$ соответствует центрам граней исходной сетки $\omega_h$, перпендикулярных пространственной координатной оси $\mathcal{O}x_\alpha$. У сетки $\omega^*_\alpha$ индекс, соответствующий этой оси, является полуцелым: $j_\alpha \hm = -1/2 + \overline{0, M}$; прочие индексы совпадают с $\omega_h$. Координаты точки на сетке для $\omega^*_\alpha$ вычисляются по той же формуле $\omega_{j_1,j_2,j_3}^k$, приведенной выше.

Далее в случае, если некоторый координатный индекс не меняется в рамках целой формулы, он будет опускаться.

Пусть $H(\omega)$~-- пространство функций, заданных на сетке $\omega$. Введем разностные операторы $\delta_\alpha: H(\omega_h) \to H(\omega^*_\alpha)$; $\delta^*_\alpha: H(\omega^*_\alpha) \to H(\omega_h)$; $\mathring{\delta}_\alpha: H(\omega_h) \to H(\omega_h)$. Считая $j_\alpha$ целым,
\[
    [\delta_\alpha \zeta]_{j_\alpha + 1/2} = \cfrac{\zeta_{j_\alpha + 1} - \zeta_{j_\alpha}}{h}, \quad [\delta^*_\alpha \eta]_{j_\alpha} = \cfrac{\eta_{j_\alpha + 1/2} - \eta_{j_\alpha - 1/2}}{h}, \quad [\delta^*_\alpha \zeta]_{j_\alpha} = \cfrac{\zeta_{j_\alpha + 1} - \zeta_{j_\alpha - 1}}{2h}.
\]
Заметим, что $\mathring{\delta}_\alpha = \delta^*_\alpha \circ \delta_\alpha$. Все три оператора~-- сеточные приближения дифференцирования $\partial / \partial x_\alpha$ вдоль оси $\mathcal{O}x_\alpha$.

Также в ряде случаев при построении разностной схемы будет нужно переводить функцию с сетки $\omega_h$ на $\omega^*_\alpha$. Для краткости обозначений введем операторы
\[
    [s_\alpha \zeta]_{j_\alpha + 1/2} = \cfrac{\zeta_{j_\alpha + 1} - \zeta_{j_\alpha}}{2}, \qquad [g_\alpha \zeta]_{j_\alpha + 1/2} = \left( \cfrac{2}{1 / \zeta_{j_\alpha + 1} + 1 / \zeta_{j_\alpha}} \right)^{-1}.
\]
Функции на сетке $\omega^*_\alpha$ получаются осреднением значений исходной функции: в смысле среднего арифметического ($s_\alpha$) либо геометрического ($g_\alpha$).

\begin{subequations}
\label{sch:complete}
Перейдем к описанию схемы расчета. На каждом временном слое вначале решается разностный аналог уравнения \eqref{eq:electrostatic_c}
\begin{equation}
    \Div_h (\epsilon[\phi^k] \cdot \nabla_h \Phi^{k + 1}) = -\rho^k
    \label{sch:complete_Phi}
\end{equation}
относительно нового значения электрического потенциала $\Phi^{k + 1}$. Здесь $\Div_h$ и $\nabla_h$ обозначены сеточные приближения дифференциальных операторов, конкретный вид которых сейчас неважен. Решение уравнения происходит итерационно, многосеточным методом.

Далее делается шаг явной схемы для функций $\rho$ и $\phi$ по разностным аппроксимациям уравнений \eqref{eq:electrostatic_b} и \eqref{eq:phi}. Производную $\partial / \partial t$ в обоих случаях приблизим простейшей разностной производной первого порядка. Оператору $\Div$ везде будет соответствовать сумма $\delta^*_\alpha$; градиенту под знаком $\Div$~-- $\delta_\alpha$. $\sigma[\phi^k]$ в уравнении \eqref{eq:electrostatic_b} необходимо перевести на~$\omega^*_\alpha$~-- добьемся этого взятием среднего геометрического $g_\alpha$. Операции $(\nabla \zeta, \nabla \zeta) \equiv \| \nabla \zeta \|_2^2$ сопоставим
\begin{equation}
    \| \zeta \|_h^2 = \sum\limits_{\alpha = 1,2,3} (\mathring{\delta}_\alpha \zeta)^2.
    \label{sch:norm_squared}
\end{equation}
В уравнении \eqref{eq:phi} она встречается дважды: для $\Phi$, где описанной конструкции достаточно, и для $\phi$ под знаком $\Div$, где дополнительно требуется перевод функции на $\omega^*_\alpha$, что сделаем оператором среднего арифметического $s_\alpha$. Таким образом, получим разностные уравнения
\begin{gather}
    \cfrac{\rho^{k + 1} - \rho^k}{\tau} = \sum\limits_{\alpha = 1,2,3} \delta^*_\alpha(g_\alpha \sigma[\phi^k] \cdot \delta_\alpha \Phi^{k + 1}),
    \label{sch:complete_rho} \\
    \cfrac{\phi^{k + 1} - \phi^k}{\tau} = \half \epsilon'(\phi^k) \| \nabla \Phi^{k + 1} \|_h^2 + \cfrac{\Gamma}{l^2} f'(\phi^k) + \sum\limits_{\alpha = 1,2,3} \delta^*_\alpha \left( \cfrac{\Gamma}{2} \delta_\alpha \phi^k + \beta \Gamma l^2 \cdot s_\alpha \| \nabla \phi^k \|_h^2 \cdot \delta_\alpha \phi^k \right),
    \label{sch:complete_phi}
\end{gather}
откуда легко выражаются $\rho^{k + 1}$ и $\phi^{k + 1}$.
\end{subequations}

На границе области $\clOmega$ заданы граничные условия функций $\phi$ и $\Phi$ типа Дирихле либо однородные типа Неймана, согласно \eqref{cond:boundary_basic}. Считаем, что $\Phi|_{y = L} = \Phi^- = 0$; $\Phi|_{y = 0} \hm = \Phi^+$ в текущей постановке задачи принимается равным константе (условие \eqref{cond:boundary_voltage}, типа Дирихле), однако может рассматриваться и как числовая свободная переменная модели (условие \eqref{cond:boundary_balance} или \eqref{cond:boundary_charge}).

Для аппроксимации граничных условий на каждой грани $\clOmega$ добавим к сетке $\omega_h$ по одному слою фиктивных ячеек (с индексами $-1$, $M$ по соответствующей координате). Теперь разностная производная $\delta_\alpha$ определена на гранях с $j_\alpha = -1/2, M - 1/2$; производная $\mathring{\delta}_\alpha$~-- в ячейках с $j_\alpha = 0, M - 1$. Для условия Дирихле $\phi|_{x_\alpha = 0, L} = \psi$ примем $\phi^k_{j_\alpha = -1,M} = \psi(\omega_{j_\alpha = -1,M})$; для однородного условия Неймана перед шагом явной схемы проведем присваивание $\phi^k_{j_\alpha = -1,M} := \phi^k_{j_\alpha = 0, M - 1}$, так чтобы $[\delta_\alpha \phi]_{j_\alpha = -1/2, M - 1/2}^k = 0$.

В исследуемых далее постановках на <<боковых>>, перпендикулярных осям~$x$ и~$z$, гранях $\clOmega$ задано условие типа Дирихле $\phi = 1$ и однородное типа Неймана для $\Phi$; на~гранях $y = 0$ и $y = H$~-- однородное условие Неймана для $\phi$ и условие Дирихле для~$\Phi$ равенства $\Phi^+$ и $0$ соответственно.


\subsection{Адаптация шага по времени}

Раздел \ref{sec:adaptation} посвящен адаптации расчетного шага по времени в исследуемой модели, которая тогда была проведена для упрощенной, одномерной, постановки задачи. Однако выбранные методы не зависят от размерности, и основной целью изначально было внедрение адаптации в программу для моделирования полной постановки задачи.

В описанную программу, реализующую вычислительную схему \eqref{sch:complete}, введен метод адаптации расчетного шага по времени по фазовому полю, наиболее простой из~трех рассмотренных в разделе \ref{sec:adaptation} и показавший в целом лучшие результаты в программе для~одномерного случая задачи.

В уравнениях \eqref{sch:complete_rho}, \eqref{sch:complete_phi} шаг по времени $\tau$ заменяется на переменный $\tau^k$, рассчитываемый по формулам \eqref{sch:time_step_min_max}, \eqref{sch:time_step_phi}. В качестве $[\partial \phi / \partial t]_h$ в знаменателе выражения~\eqref{sch:time_step_phi} берется значение левой части формулы \eqref{sch:complete_phi}~-- так как в расчете уже используется ее правая часть, дополнительных вычислений это не потребует. Однако из-за распределения счета между вычислительными узлами MPI использовать при переходе между временными слоями $k$ и $k + 1$ текущее значение производной затратно: необходима лишняя синхронизация процессов с массовой операцией максимума. Поэтому будет использоваться значение производной с предыдущего перехода, а именно:
\begin{equation}
    \widetilde{\tau}^k = \tol \cdot \left( \max\limits_{j_{1,2,3}} \left| \cfrac{\phi^k - \phi^{k - 1}}{\tau^{k - 1}} \right|_{j_{1,2,3}} \right)^{-1}.
    \label{sch:time_step_phi_complete}
\end{equation}


\subsection{Параметры расчетов в модельных постановках}

Проведены два расчета в модельных постановках в программе, реализующей вычислительную схему \eqref{sch:complete}. Перечислим параметры модели.

В обоих расчетах размер пространственной сетки по каждой из координат $M \hm = L / h = 129$ плюс два слоя фиктивных ячеек, по одному на каждой грани $\clOmega$. Из-за особенностей многосеточного метода, применяемого для решения уравнения \eqref{sch:complete_Phi}, $M$~должно иметь вид $2^k + 1$.

Параметры расчетов перечислены в таблице \ref{tab:parameters_complete}. Константы модели приближенно описывают физический эксперимент; они имеют те же порядки, что в таблице \ref{tab:parameters_one_dim}, относящейся к расчетам одномерной программы для упрощенной постановки задачи. По~сравнению с таблицей \ref{tab:parameters_one_dim} в $2$ раза увеличено $l$ и в $4$ раза~-- $\Gamma$, что соответствует росту характерной толщины диффузной границы в $2$ раза при сохранении прочих свойств системы, таких как, например, устойчивость неповрежденной среды (см. \eqref{char:equilibriums_voltage}).

Для среды, близкой к неповрежденной, имеет место приближение $E = \Phi^+ / L \hm = 5 \cdot 10^7 \; \unitV$. В таком случае характеристическая величина $\xi$, заданная формулой \eqref{char:equilibriums_voltage}, для~перечисленных параметров модели равна $80.895$, что выраженно соответствует случаю <<сильного напряжения>> $\xi > (1 + \delta)^2$, означая неустойчивость состояния $\phi = 1$ и~развитие канала пробоя из сколь угодно малых его возмущений.

Проверим выполнение для используемых параметров модели условия устойчивости \eqref{cond:stability_voltage} явной разностной схемы. Первый аргумент минимума (с учетом $m^{-1}$) равен $2.035 \cdot 10^{-7} \; \units$~-- неравенству с ним удовлетворяют и $\taumin$, и $\taumax$. Второй аргумент, даже если принять <<мягкую>> оценку $E = \Phi^+ / L$ для случая неповрежденной среды, оказывается равен $4.829 \cdot 10^{-12} \; \units$. Если же использовать для $E$ строгую оценку сверху $E \leqslant \Phi^+ / h$, то вовсе получается значение $2.902 \cdot 10^{-16} \; \units$. Таким образом, $\taumin$ превышает значение правой части условия \eqref{cond:stability_voltage} на порядки. Это одно из проявлений значимой проблемы недостаточного разрешения дискретной модели при работе с исследуемой задачей. Вероятно, она возникает из-за <<жесткости>> интерполирующих функций $\epsilon(\phi)$, $\sigma(\phi)$, однако, в свою очередь, выбор существующей их формы не беспочвенен. Автор высказывает предположение, что проблема значительно смягчится при использовании модели с переменным $\Phi^+$, как, например, в постановках \eqref{cond:boundary_balance} или \eqref{cond:boundary_charge} либо при учете в модели внешней электрической цепи.

Впрочем, на текущем этапе работы удовлетворимся следующим несложным решением проблемы: при возникновении $\phi_{j_1,j_2,j_3}^k < 0$ будем немедленно проводить присваивание $\phi_{j_1,j_2,j_3}^k := 0$.

Как указано в таблице \ref{tab:parameters_complete}, $\delta = \delta_\sigma = 10^{-3}$. Это означает, что $\sigma / \epsilon = k = \Const$. Это условие было ранее включено в постановку \eqref{cond:boundary_voltage} и с учетом $\rho(\vx, 0) = 0$ влечет $\rho \equiv 0$, позволяя не рассматривать динамику свободных зарядов в системе. В силу выбранного упрощения, в таблице \ref{tab:parameters_complete} не указано значения $\sigma_0$~-- оно не играет роли.

В формуле \eqref{sch:time_step_phi_complete} адаптивного шага по времени используем $\tol = 10^{-2}$. Интуитивно это означает, что за один шаг по времени, если позволит ограничение через $\taumin$, изменение фазового поля $\phi$ в каждой точке будет происходить не более чем на $10^{-2}$.


\subsection{Результаты расчетов}

Начальное условие первого расчета для функции $\phi$ есть
\[
    \phi(\vx, 0) = \phi_0(\vx) = \begin{cases}
        0 \text{ при } \vx \in \left[ \cfrac{63}{129} L, \cfrac{65}{129} L \right]_x \times \left[ \cfrac{119}{129} L, L \right]_y \times \left[ \cfrac{63}{129} L, \cfrac{65}{129} L \right]_z; \\
        0.999 \text{ иначе}.
    \end{cases}
\]
Условие соответствует проводящей <<иголке>>, прикрепленной к середине верхней обкладки. В остальной расчетной области значение $\phi$ уменьшено на $10^{-3}$ относительно неповрежденного вещества, что выполняет роль начального малого возмущения равновесного состояния среды.

Результаты расчета изображены на рис. \ref{fig:needle_result}. Расчет занимает $51000$ временных слоев сетки и $7.019 \cdot 10^{-7} \; \units$ времени модели. Окрашены ячейки сетки, для которых $\phi \leqslant 0.03$. Функция начального условия $\phi_0(\vx)$ разрывна; однако для модели с оператором Лапласа это не представляет проблем: решение <<разглаживается>> на первых временных слоях. Канал пробоя начинает расти в длину и в толщину; из четырех <<углов>> начальной конфигурации развиваются четыре ветви, которые расходятся к краям образца. Постепенно сами ветви начинают расщепляться на отдельные отростки, пока, наконец, все ускоряя рост, канал не доходит до нижней обкладки. Его итоговый вид представлен подробно на рис. \ref{fig:needle_result_3}. Две дальние от зрителя ветви симметричного канала изображены полупрозрачно для удобства.

В качестве начального условия второго расчета взяты независимые случайные возмущения состояния $\phi = 1$ в каждой ячейке сетки:
\[
    \phi(\vx, 0) = \phi_0(\vx) \sim 1 - \text{Exp}(10^3).
\]
Здесь $\text{Exp}(\lambda)$~-- экспоненциальное распределение с плотностью вероятности $f_\text{Exp}(x) \hm = I\{x \geqslant 0\} \cdot \lambda e^{-\lambda x}$, математическим ожиданием $\lambda^{-1}$ и дисперсией $\lambda^{-2}$.

Результаты расчета изображены на рис. \ref{fig:disturbance_result}. Расчет занимает $22000$ временных слоев сетки и $1.715 \cdot 10^{-6} \; \units$ времени модели. Для двух описанных расчетов также построены графики двух интегральных величин: среднего фазового поля $\phi$ по области $\Omega$ и свободной энергии $\Psi$. Плотность $\psi$ свободной энергии определяется выражением \eqref{eq:energy_density_free}. Использована следующая разностная аппроксимация:
\begin{gather*}
    \psi^k = \cfrac{\Gamma}{l^2} [1 - f(\phi^k)] + \cfrac{\Gamma}{4} \| \nabla \phi^k \|_h^2 + \cfrac{\beta \Gamma l^2}{4} \| \nabla \phi^k \|_h^4, \\
    \Psi^k = \sum\limits_{j_{1,2,3}} h^3 \psi_{j_1,j_2,j_3}^k.
\end{gather*}
Разностный оператор $\| \cdot \|_h^2$ задан формулой \eqref{sch:norm_squared}.

Поведение указанных двух интегральных величин для расчетов с проводящей иголкой и со случайными возмущениями неповрежденной среды изображено соответственно на рис. \ref{fig:needle_integral_values} и \ref{fig:disturbance_integral_values}. В расчете с проводящей иголкой, как видно из рис. \ref{fig:needle_phi_mean}, уменьшение среднего фазового поля $\phi$ на протяжении опыта близко к равномерному, что свидетельствует в пользу примененного метода адаптации шага по времени. При этом в правой части графика \ref{fig:needle_energy} свободной энергии $\Psi$ наблюдается явная аномалия: около момента пробоя образца <<насквозь>>, немногим после прохождения системой конфигурации на рис. \ref{fig:needle_result_3}, происходит резкое (и негладкое) падение $\Psi$, а вскоре после этого~-- такого же вида рост. Как обсуждалось ранее, согласно замечанию \ref{rem:Phi_constant} в исследуемой постановке задачи с~$\Phi^+ \hm = \Const$ и $\rho \equiv 0$ свободная энергия $\Psi$ в процессе эволюции системы не возрастает. Это свойство нарушается в указанной части графика. Названную проблему следует изучить в будущем; автор предполагает, что она возникает из-за не вполне верной программной обработки плотности свободных зарядов $\rho$ и, таким образом, нарушения свойства $\rho \equiv 0$ в расчете.

Поведение интегральных величин расчета со случайными возмущениями среды оказывается куда более удачным. Кажущееся резкое падение среднего фазового поля~$\phi$ в самом начале графика \ref{fig:disturbance_phi_mean} на деле происходит с контролируемой производной не~более $10^{-2}$, что соответствует использованному значению $\tol$. На графике \ref{fig:disturbance_energy} видно постепенное убывание свободной энергии $\Psi$, как и должно быть согласно теории. Примечательно, что ее относительное падение в наиболее существенной и интересной нам части опыта, моменты которой изображены на рис. \ref{fig:disturbance_result}, совсем невелико; куда большее убывание $\Psi$ происходит уже после первого <<сквозного>> пробоя, при крайнем разрушении образца.

Описанные расчеты проведены на вычислительном комплексе K60, в секции CPU. В каждом случае использовался параллельный счет с помощью MPI на $56$ти процессорах двух узлов комплекса. Расчеты в тех же конфигурациях помимо этого проведены без использования адаптации шага по времени.

Отметим, что для расчетной сетки по времени не вводилось никакой особой структуры, как, например, это делалось в программе для упрощенной, одномерной, постановки задачи (подробности в подразделе \ref{ssec:grid_adaptive_structure}); напрямую использовался шаг $\tau^k$, получаемый по формулам \eqref{sch:time_step_min_max}, \eqref{sch:time_step_phi_complete}. Для сравнения результатов расчетов с адаптацией и без нее автор в настоящий момент ограничился проверкой качественного соответствия процессов в системе.
