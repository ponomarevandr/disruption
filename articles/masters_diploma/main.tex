\begin{abstract}
    \noindent
    Предмет исследования настоящей работы -- математическая модель типа диффузной границы, описывающая развитие канала электрического пробоя в твердом диэлектрике. Цель работы -- предложить решение ряда вопросов различной природы, как теоретических, так и связанных с практикой моделирования задачи. Вначале дано описание математической модели; выведены балансовые соотношения для различных функционалов энергии системы, приведено несколько вариантов постаноки граничных условий; исследована устойчивость положений равновесия системы. Далее проделан анализ явной разностной схемы для упрощенной постановки задачи, выведены условия устойчивости схемы при различных типах граничных условий. Затем исследовано несколько методов адаптации расчетного шага по времени для используемой модели, один из которых предложен автором настоящей работы. Наконец, описано внедрение адаптации шага по времени в разрабатываемую научным коллективом программу для моделирования полного, трехмерного, случая задачи; проведены тестовых два расчета в модельных постановках, представлены и проанализированы их результаты.
\end{abstract}
